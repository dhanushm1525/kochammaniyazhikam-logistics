# Kochammaniyazhikam Logistics

Welcome to the **Kochammaniyazhikam Logistics** web application project! This is a modern, responsive, and performance-optimized platform designed to showcase a comprehensive suite of logistics, transportation, and equipment hiring services. Built primarily with React, Vite, and Tailwind CSS, it caters to all client supply-chain needs.

## 📁 Application Structure & Pages

The application is structured into the following key pages to provide a holistic user experience:

- **Home Page (`Home.jsx`):** The primary landing area that hooks the visitor with a high-impact Hero section, outlines core features, and serves as a gateway to all our featured logistics services.
- **About Us (`AboutUs.jsx`):** A dedicated section that chronicles the mission, vision, and robust history behind Kochammaniyazhikam Logistics.
- **Services Directory (`ServicesPage.jsx`):** A centralized hub listing our 10 primary service domains including road transport, machinery hiring, and value-added services.
- **Service Detailed View (`ServiceCategoryPage.jsx`):** A dynamic, data-driven page rendering specific details, sub-services, and imagery for any chosen service category (e.g., Aggregates Supply Services, ODC Transport, Heavy Equipment Towing).
- **FAQ Page (`FAQPage.jsx`):** A comprehensive help center answering common questions about our policies, operations, shipping updates, and service agreements.
- **Contact Page (`ContactPage.jsx`):** The communication bridge offering direct pathways to our team through verified contact details (incl. primary support phones and WhatsApp), location details, and a responsive contact form.

## 🏢 Services Offered

Our digital platform details the following core offerings:
1. **Supply Chain Management** (Route planning, fleet management, vendor coordination)
2. **Specialized Transport Services** (Project cargo, break bulk, industrial relocation)
3. **Value Added Services** (GPS tracking, cargo insurance, 24/7 customer support, E-way bill processing)
4. **Heavy Equipment Towing** (Low bed, semi lowbed, multi-axle trailers)
5. **ODC Transport & Heavy Equipment** (Excavator transport, bulldozers, crane transport)
6. **Machinery & Excavator Hiring** (Rentals for crushers, breakers, diggers, and excavators)
7. **Road Transport** (Full load trucks, long-distance interstate transport, Lorry, Taurus)
8. **Aggregates Supply Services** (M-Sand, P-Sand, Crushed Stone, River Sand, WMM Materials)
9. **Heavy Crane Operations** (Up to 500 TON All Terrain Crane availability)

## 🛠 Tech Stack

- **Frontend Framework:** React 19
- **Build Tool:** Vite
- **Styling:** Tailwind CSS, Framer Motion (for dynamic UI animations)
- **Routing:** React Router DOM (for seamless Single Page App transitions)
- **Icons & UI:** Lucide React, Embla Carousel

## 🚦 Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start the development server:**
   ```bash
   npm run dev
   ```

3. **Build for production:**
   ```bash
   npm run build
   ```
